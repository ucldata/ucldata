Workshop 2 2017 Friday 17th November 2017

Location: Drayton House B20 Jevons LT, UCL
Time: 18:00 - 20:00, Friday 17th November 2017

What you need: A computer with the most recent version of Python 3 installed, we recommend the Anaconda distribution.

Welcome to the Data Science Curriculum - UCL's student-led machine learning workshop series. Throughout the academic year, we'll bring you 8 fully fledged, 2-hour workshops sessions where you'll learn everything from coding in R or Python, statistical thinking, and advanced machine learning algorithms.

In our second workshop, you can expect to be introduced to pandas library, a powerful library for data manipulation and analysis, which provides an easy to use data object. From here, we will teach you how to use sklearn for basic regression on the built-in Boston Housing dataset and learn statistical thinking along the way.
